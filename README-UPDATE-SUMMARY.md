### Quick Screenshot Process:

1. Create screenshots directory:
   ```bash
   mkdir -p screenshots
   ```

2. Open the helper:
   ```bash
   open screenshot-helper.html
   ```

3. Take screenshots as guided:
   - desktop-main.png (1920x1080)
   - desktop-attention.png
   - desktop-code.png
   - mobile-content.png (375x812)
   - mobile-viz.png (375x812)

4. Copy the template from `screenshot-template.md` into README.md

5. Commit and push:
   ```bash
   git add screenshots/ README.md
   git commit -m "Add screenshots to README"
   git push
   ```

## 🎯 Screenshot Tips

### Desktop Screenshots (Chrome/Firefox):
1. Open https://yunqing.github.io/microgpt_lab/
2. Press Cmd+Shift+4 (macOS) or use Snipping Tool (Windows)
3. Capture the full window

### Mobile Screenshots (Chrome DevTools):
1. Open DevTools (F12)
2. Click device icon (Cmd+Shift+M)
3. Select "iPhone 12 Pro"
4. Click ⋮ → "Capture screenshot"
